    • Defined in index.ts:118

    Dispose of the set and dispose the items it contains.

    Notes

    Items are disposed in the order they are added to the set.

    It is unsafe to use the set after it has been disposed.

    If this method is called more than once, all calls made after the first will be a no-op.

    Returns void
